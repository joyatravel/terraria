# terraria

_A status page for the Tugolo Terraria server._

[![Drone][drone-img]][drone]

## Setup

```bash
# Clone repository.
git clone git@github.com:tugolo/terraria

# Install dependencies and configure githooks.
cd terraria && make setup
```
